本次重构已完成从 UUID 架构到组合键架构的完整迁移，实现了以下核心功能：

### ✅ 核心架构变更

**1. 数据模型重构**

## 🔧 常见问题排查

### 问题 1: 迁移失败

**症状**: 迁移 API 返回错误

**排查步骤**:
1. 检查备份目录是否创建
2. 检查 `artists.json` 和 `image_artists.json` 文件权限
3. 查看浏览器控制台和 ComfyUI 终端错误日志

**解决方案**:
- 从备份目录恢复：`cp backup_*/artists.json artists.json`
- 检查 JSON 格式是否正确

### 问题 2: 图片不显示

**症状**: 图片显示 404 或加载失败

**排查步骤**:
1. 检查图片路径是否正确
2. 检查 `buildImageUrl()` 函数
3. 查看浏览器网络面板

**解决方案**:
- 确保图片文件存在于 `ComfyUI/output/artist_gallery/`
- 检查文件名编码

### 问题 3: 复制后图片不共享

**症状**: 复制画师后，新画师没有图片

**排查步骤**:
1. 检查 `image_artists.json` 中的 `artistNames` 数组
2. 确认两个画师的 `name` 字段完全一致

**解决方案**:
- 确保 `name` 字段完全相同（包括大小写）
- 重新执行迁移

### 问题 4: 选择器输出格式错误

**症状**: ArtistSelector 节点输出不正确

**排查步骤**:
1. 打开浏览器控制台
2. 查看是否有 JavaScript 错误
3. 检查 localStorage 中的选择数据

**解决方案**:
- 清除浏览器缓存
- 清除 localStorage: `localStorage.removeItem('artist_selector_selection')`
