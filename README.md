# 🎨 Artist Gallery - ComfyUI 画师图库插件

在 ComfyUI 中展示和管理画师图库的插件。

## 功能特点

- ✨ 悬浮可拖动的图库入口按钮（页面右下角）
- 🖼️ 在页面内弹窗中打开画师图库管理界面（非新窗口）
- 💾 按钮位置自动保存，刷新后保持位置
- 📁 自动扫描 ComfyUI output 目录下的画师图片
- 🎯 支持按画师名称搜索
- ⭐ 支持收藏画师
- 📊 支持按名称、数量、时间排序
- 📋 一键复制画师标签（@画师名）
- 🔍 支持大图预览和导航

## 图片格式要求

插件会自动识别以下格式的图片文件：

```
@画师名,_序号.扩展名
```

例如：
- `@artist_name,_1.png`
- `@artist_name,_2.png`
- `@画师名,_1.jpg`

支持的图片格式：`.png`, `.jpg`, `.jpeg`, `.webp`

## 安装方法

1. 将 `artist_gallery` 文件夹复制到 ComfyUI 的 `custom_nodes` 目录：
   ```
   ComfyUI/custom_nodes/artist_gallery/
   ```

2. 重启 ComfyUI

3. 在页面右下角会看到一个悬浮的 🎨 按钮

4. 可以拖动按钮到任意位置，位置会自动保存

## 使用方法

1. 拖动右下角的 🎨 悬浮按钮到喜欢的位置（可选）
2. 点击悬浮按钮打开图库弹窗
2. 在弹窗中查看所有画师图库
3. 使用搜索框过滤画师
4. 点击图片查看大图
5. 点击 📋 复制画师标签
6. 点击 ⭐ 收藏喜欢的画师
7. 点击 🔄 刷新数据
8. 点击 ✕ 或按 ESC 关闭弹窗

## 技术实现

- **后端**: Python + aiohttp API
- **前端**: 原生 JavaScript（无依赖）
- **样式**: 渐变色粉色系主题
- **弹窗**: 页面内模态框（非浏览器新窗口）

## API 端点

- `GET /artist_gallery/data` - 获取图库数据（JSON）

## 注意事项

- 确保图片文件命名格式正确
- 图片应放在 ComfyUI 的 `output` 目录（或子目录）下
- 支持嵌套目录扫描
- 收藏数据保存在浏览器 localStorage 中

## 故障排除

**问题**: 看不到悬浮按钮
- 确认已重启 ComfyUI
- 检查页面右下角
- 按钮可能被拖到其他位置，检查整个页面
- 打开浏览器控制台（F12）查看是否有错误

**问题**: 按钮位置不理想
- 直接拖动按钮到任意位置
- 位置会自动保存，刷新后保持

**问题**: 点击按钮没有反应
- 打开浏览器开发者控制台（F12）查看错误信息
- 检查插件是否正确安装在 `custom_nodes` 目录
- 确认已重启 ComfyUI

**问题**: 图片不显示（404 错误）
- 确认图片路径正确
- 检查图片是否在 output 目录下
- 尝试点击刷新按钮重新加载数据

**问题**: 扫描不到图片
- 确认图片文件名格式符合要求（@画师名,_序号.扩展名）
- 检查图片是否在 ComfyUI 的 output 目录中
- 查看浏览器控制台的错误信息
