# ArtistSelector 高级配置 - 快速开始

## 🚀 3分钟快速上手

### 第一步：重启和刷新
```
1. 完全关闭 ComfyUI
2. 重新启动 ComfyUI
3. 在浏览器中硬刷新（Ctrl+Shift+R）
```

### 第二步：创建测试工作流
```
1. 添加 ArtistSelector 节点
2. 选择几个画师
3. 添加 Print 节点查看输出
4. 连接节点
5. 执行工作流
```

### 第三步：配置格式化
```
1. 点击 "⚙️ 全局配置" 按钮
2. 在 "单画师格式" 中输入：({content}:1.2)
3. 点击 "保存"
4. 再次执行工作流
5. 查看输出：(artist1:1.2),(artist2:1.2)
```

## 📖 常用配置示例

### Stable Diffusion 权重格式
```
格式：({content}:{random(0.5,1.5,0.1)})
示例输出：(artist1:0.8),(artist2:1.2)
```

### 简单编号格式
```
格式：{content}_{random(1,999,1)}
示例输出：artist1_456,artist2_123
```

### 括号标记格式
```
格式：[{content}]
示例输出：[artist1],[artist2],[artist3]
```

### 高级组合格式
```
格式：({content}|{random(0.7,1.3,0.1)})
示例输出：(artist1|0.9),(artist2|1.1)
```

## 🎯 功能速查

| 功能 | 配置位置 | 说明 |
|------|---------|------|
| 全局格式化 | ⚙️ 全局配置 | 所有画师使用相同格式 |
| 分类格式化 | 分类标签旁的 ⚙️ | 某个分类使用独立格式 |
| 多画师随机 | ⚙️ 全局配置 → 多画师随机规则 | 每次随机选N个画师 |
| 循环模式 | ⚙️ 全局配置 → 循环模式 | 每次输出一个画师，循环使用 |

## ⚡ 常见问题快速解决

### Q: 配置不生效？
```
解决：1. 重启ComfyUI 2. 刷新浏览器 3. 重新配置
```

### Q: 想恢复默认？
```
解决：点击 ⚙️ 全局配置 → 格式输入 {content} → 保存
```

### Q: 分类配置如何删除？
```
解决：点击分类旁的 ⚙️ → 取消勾选"使用独立配置" → 保存
```

### Q: 循环模式如何重置？
```
解决：暂时没有UI按钮，需要重新选择画师或重启服务器
```

## 🎨 格式字符串语法

```
{content}              → 画师名称
{random(1,100,1)}      → 1-100之间的整数
{random(0.5,1.5,0.1)}  → 0.5-1.5之间的浮点数（步长0.1）
```

**组合使用**：
```
({content}:{random(0.7,1.3,0.1)})
↓
(artist_name:0.9)
```
